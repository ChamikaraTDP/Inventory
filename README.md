## Environment
1. php 7.4
2. node 14
3. php composer
4. mysql ?
5. 

### Installation

1. Rename env.example file to .env and configure to your environment
2. composer update
3. npm install
4. php artisan key:generate
5. npm run dev
6. php artisan migrate
7. php artisan db:seed

then to run the app,  **php artisan serve**
